About naviGITor

With naviGITor, every time a team member makes a local commit, or checks out to a different branch, each team member will be notified of their actions in real time. NaviGITor is an electron-based desktop app that lets you keep track of work being done on any repo within an organization. NaviGITor also provides full git functionality within the app itself using native node processes.
